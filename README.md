# Brutus
A brute forcing tool for multiple services and websites

## Getting Started

### Requirements

Pyfiglet:
    
    pip install pyfiglet

Mechanize

    pip instal mechanize
    
Paramiko

    pip install paramiko

### Using the tool

        python main.py
