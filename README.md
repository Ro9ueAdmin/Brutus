# Brutus
A brute forcing tool for multiple services and websites

## Getting Started

### Requirements

Pyfiglet:
    
    pip install pyfiglet
    
Paramiko

    pip install paramiko
    
ftplib
    
    <Already present>

Script: To install all the dependencies-
    
    chmod +x install-requirements.sh
    ./install-requirements.sh

### Using the tool

        python main.py
